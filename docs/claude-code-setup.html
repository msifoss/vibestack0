01

Create an Anthropic Account

Anthropic is the company that makes Claude. You need a free account to get started.

Go to claude.ai

Open your browser and go to claude.ai. Click Sign up.

Sign up

You can sign up with your Google account or with an email and password. Either works ‚Äî pick whichever is easier for you.

Verify your email

If you signed up with email, check your inbox for a verification email from Anthropic. Click the link to verify your account.

02

Subscribe to Claude

Claude Code requires a paid subscription to use. A subscription gives you access to Claude Code directly through your account ‚Äî no API keys needed.

Choose your plan

Go to your Claude settings and choose a subscription tier. Here's what's available:

PlanClaude Code AccessBest For
Pro ($20/mo)IncludedGetting started, light usage
Max ($100/mo)Included, higher limitsHeavy daily usage
Start with Pro The Pro plan is great for getting started. You can always upgrade to Max later if you find yourself hitting usage limits.
Subscribe

On claude.ai, click your name or profile icon, then go to Settings ‚Üí Subscription. Choose a plan and enter your payment details.

Alternative: API keys (advanced)

If you prefer usage-based pricing instead of a subscription, you can use API keys. This is more complex and typically used by developers who want fine-grained control over costs.

API key setup For API key setup, see the Anthropic API documentation. You'll need to create an API key at console.anthropic.com and add billing credits. Most beginners should use the subscription method above instead.
03

Authenticate Claude Code

Now that you have a subscription, let's connect Claude Code to your account. This uses a simple browser-based login ‚Äî no API keys to copy or paste.

Open your terminal in VS Code

Open VS Code, then open the integrated terminal:

How to open the terminal Press Ctrl+` (Windows) or ‚åò+` (Mac) ‚Äî that's the backtick key, usually above Tab. A terminal panel opens at the bottom of VS Code.
Run the claude command

Type claude and press Enter:

Terminal (in VS Code) 
claude
Follow the OAuth flow

Claude Code will detect that you're not logged in and start the authentication process:

What happens 1. Claude Code prints a message saying it needs to authenticate
2. Your browser opens automatically to an Anthropic login page
3. Log in with the same account you subscribed with
4. Click Allow to authorize Claude Code
5. The browser shows a success message ‚Äî you can close the tab
6. Back in your terminal, Claude Code confirms it's authenticated
Browser didn't open? If your browser doesn't open automatically, Claude Code will display a URL in the terminal. Copy that URL and paste it into your browser manually.
04

Verify It Works

Let's make sure everything is connected and working.

Check the version

Confirm Claude Code is installed and accessible:

Terminal 
claude --version

This should print a version number like 1.x.x.

Try a simple prompt

Test that Claude Code can communicate with the API:

Terminal 
claude "Say hello in one sentence"

Claude should respond with a friendly greeting. If you see a response, you're fully connected!

Start coding with Claude

You're all set. Here's how to use Claude Code in your projects:

Quick reference Interactive mode: Run claude in your project folder to start a conversation
One-shot: Run claude "your question here" for quick answers
Help: Run claude --help to see all available commands
claude --version
claude "hello"
claude --help
